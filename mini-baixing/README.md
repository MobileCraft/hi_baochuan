# mini baixing

## 作用
- 快速了解 Baixing iOS 项目结构、常用组件
- 尽量少的代码完成百姓网客户端基本功能

## 功能集（业务）
- Category
	- 分级显示一二级类目
	- 一级类目显示 icon、类目名称
	- 二级类目显示类目名称
- Listing
	- 按时间顺序显示 ads
	- 可加载更多
	- 可重新加载
	- 每个 ad 显示 标题、图片、时间
- viewad
	- 显示 标题、`metas`、描述
	- 可拨打电话
	- 可查看大图

## 知识点（技术）
- Cocoapods
- Baochuan data
- ad 结构
- 手势返回
- BCConfig

## 实现
- `master` 分支为 mini-baixing 初始化状态
- `staging` 分支为 mini-baixing 完成功能状态
- `staging v1.0` 由 mobile team 共同修订完成
- 后续 _完成 mini-baixing 者_ 有资格 __提议__ 改进 `staging`
